Use the `get_report` tool to generate a deep report on the requested topic.

The report includes:
- Overview and current state of the topic
- Key data points and recent developments
- Analysis from multiple angles
- Conclusions and actionable insights

Extract the topic from the user's message. If no topic is specified, ask the user what they want a report on.
