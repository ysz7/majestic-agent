Capabilities

Built Different.

Clean architecture. No 10,000-line files. Every component has one job.

01
Universal Execution

Give it any task in plain language. The agent picks the right tools and executes — no command menus.

agentic-loop
02
Persistent Memory

Remembers preferences, past sessions, and learned patterns across every conversation and platform.

cross-session
03
Universal Search

One query across news, docs, market data, and all past conversations. Hybrid FTS5 + vector.

sqlite-vec · fts5
04
MCP Integration

Any MCP server becomes agent tools instantly. One config line — no Python code.

mcp · stdio · sse
05
Cron Automations

"Every Monday at 9am, send me a market briefing" — plain language, delivered to any platform.

apscheduler
06
REST API

POST /chat, GET /health, GET /sessions — connect any frontend, script, or service. Optional API key auth.

http · no-dep
07
4 LLM Providers

Anthropic, OpenAI, OpenRouter, Ollama. Switch anytime with majestic model or /model.

anthropic · openai · openrouter · ollama
08
Agent Specialization

Set role and tool whitelist in config.yaml — fork for any domain without touching source code.

agent.role · tools_enabled
09
Plug-in Tools

Drop a file in tools/ and the agent picks it up automatically. No wiring, no config changes needed.

auto-registry
10
Web Dashboard

Browser UI with real-time chat, memory editor, skill CRUD, user tables, token monitoring and schedules.

react · shadcn · sse
